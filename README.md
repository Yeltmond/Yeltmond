## 🚀 Proyectos Destacados

### 🛍️ **MKPLACENI**
![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?logo=typescript&logoColor=white)
![Next.js](https://img.shields.io/badge/Next.js-000?logo=next.js&logoColor=white)
![Firebase](https://img.shields.io/badge/Firebase-FFCA28?logo=firebase&logoColor=black)

Plataforma marketplace empresarial completa. Sistema integral de e-commerce con gestión de vendedores, catálogos, carrito de compras, pagos integrados con PayPal y panel administrativo avanzado.

**Características:** 
- Autenticación y autorización con Firebase
- Gestión de productos con drag & drop
- Mapas interactivos (Google Maps)
- Procesamiento de imágenes (Sharp)
- Dashboards con análisis de datos (Recharts)
- Pagos con PayPal integrado
- UI responsive con Radix UI

**Stack:** Next.js 16 | TypeScript (97.7%) | Firebase | Zustand | TailwindCSS | Deployed en Vercel  
**Link:** [https://mkplaceni.vercel.app](https://mkplaceni.vercel.app)
